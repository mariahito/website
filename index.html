EKG Design

Fundamemtals for Electrical and Computer Engineering III

The printed circuit board seen in the first picture is the majority the board. The first stage is an instrumentation amplifier which extracts the differential from two of the leads. A shield driver helps remove some of the noise from the input. After the instrumentation amp a 4th order butterworth filter removes any of the high frequency content from the input as well and provides anti-aliasing.The output was then fed through an averaging filter on the msp430.

Shortest Path Robot

Mechatronics

Basic robot with line following cablities and able to follow and execute basic instructions of forward, left, right, u-turn, and stop. We had a webpage in which a user can enter a start node and an end node. Our server would then run dijkstra's algorithm via a python scrip behind the scenes and spit out the resultant list of nodes to visit. Then the python script would translate this into directions for the robot. Once the robot reached the destination then a new location could be entered.

Two Factor NFC Authentication

REU Research

Me and my partner worked over the summer designing a two-factor authentication system using NFC. We replaced the standard logging credentials to allow the user enter a passcode and verify its credentials by scanning a previously registered NFC tag. To make it more secure we encrypted the user's passcode and UID of the tag.

  • Client: Finish
  • Category: Identity

Home Alarm System

Mechatronics

This system uses a micro-controller that detect when the door has been open without entering the designated code. If the door is pried then the system would emit a noise and the red LED will flash. When the system is armed the red LED will remain on, if the system is not activated the green LED will remain on.
